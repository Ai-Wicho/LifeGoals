# LifeGoals - Angular & Firebase Project

Proyecto final para el curso de Fundamentos de Frontend con Angular.
Esta aplicación permite organizar tus metas personales persistentes en la nube con una prueba en firebase

## Tecnologías
* Angular 18
* Firebase (Firestore)
* Docker
* Desplegado en Render
